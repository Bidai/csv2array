# csv2array
将csv格式的字符串转换为string[][]格式的对象

## csv格式：
行与行之间以\n或者\r\n分隔
行内单元格之间以  ,  分隔
双引号表示内部为一整个字符串，内部的双引号用2个双引号表示

## 测试示例：
```C#
输入的csv字符串为：
123,456,789,"0,1"

,,,a,b,'c',"""d"""
识别结果
行数=3
内容如下：
------------------------------
|123    |456    |789    |0,1
------------------------------

------------------------------
|       |       |       |a      |b      |'c'    |"d"
```
